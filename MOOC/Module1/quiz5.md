## Quiz 5
### Tags and indexing software to help you find your way around

1. What are the limitations of the search functionality of text editors ?

    Right answer: They only work with text files
    Right answer: They work on a single file at a time

    When we have a single text file, we just need to click on the "Search" menu, type in the word we are looking for and see where it is located in that file. However, this only works when we have a single document or a single type of document.

    Also, often we need to work with text, image or data files. For example, we want aset of digital files to be used for a study can be indexed. The problem with text editors is that they can't work with all these types of data at the same time.

2. What is DocFetcher ?

    Right answer:  A cross-plateform software
    Right answer:   A desktop search engine

    DocFetcher is a **desktop search engine** available on all classic platforms: **Linux, Windows, MacOS** (cross-platform).

3. What does it make sense to use tags and keywords ?

    Right answer: To filter out overabundant information
    Right answer: To find quickly relevant information

    Tags and keywords can be used as metadata to exdescribe index information. They are used for easier and faster retrieval by searching for a specific relevant term. Also, they can indirectly help filter out overabundant information.


4. Text files are the only files to which tags and keywords can be added.

    Right answer:  False

    No, we can add keywords also to images, videos, audio files or other types of data. They can be added to any type of data as its metadata.
