## Quiz 4
### Sustainability and scalability of notes with version control

1. LibreOffice makes the comparison of two successive versions possible.

    Right answer:  True

    In teh video, it can be seen how the two colors of the crossed-out zones correspond to two different authors. Also, we can see the added and deleted parts. Moreover, when moving the mouse over text, we can see who made changes and when. 

2. A wiki engine allows us to modify a single page at a time.

    Right answer:  True

    By clicking on the "recent changes" link, all the modified pages of wiki and their modification dates are displayed. It is also practicall for viewing changes, but problems are similar as with LibreOffice: files are saved by the user (it is not handled by the wiki), andonly one page can be modified at once.

3. GitHub and GitLab let us work with binary files like images.

    Right answer:  True

    A major benefit of software such as Git is that it allows us to simultaneously correct several files. Also, we can add image files and at the same time change the text file describing the image. So, yes, Git lets us work with binary files like images.
