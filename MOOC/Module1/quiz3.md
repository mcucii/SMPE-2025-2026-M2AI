## Quiz 3
### From text files to lightweight markup languages

1. What is a text file ?

    Right answer:  A file made up (stored as) UTF-8 characters

    In general, a text file displays something that is readable, more specifically - characters included in a text file are typically encoded in UTF-8, which means that it is always possible to read such files using a text editor.

2. What is a tag ?

    Right answer: A character, or series of characters, used to structure a document that will be invisible to the final reader. 

    Tags are used in markup languages, for example, in HTML we use tag such as < h1 > to write headings.


3. Markdown is a markup language:

    Right answer:   "Light"

    Lightweight markup languages are type of markdown languages that use simpel syntax. They are easy to read even when not formatted. Markdown is a concrete example of such a language.
