### Getting started with Jupyter tool

1. What does an environment like Jupyter provide in comparison to working in the Python console or running R scripts directly?

    Right answer: It provides a well-structured history of the analyses performed.
    Right answer: It allows you to inspect data, keep a history of this inspection, and explain the transformations you perform as you go along.
    Right answer: It saves intermediate results, whether textual or graphical.
    Right answer: It allows you to generate documents in HTML or PDF.
    Right answer: It allows you to ensure that a figure is the result of the computation described in the document.


2. In Jupyter, what features are provided for the Python language but not available for the R language? 

    Right answer: There are the same features for both languages

    I tried out all of the given points, and realized both languages provide same features: autocompletion on variable names, the non-persistence of variables and their state, syntax highlighting, it is possible to execute both Python code and R code at the same time; for both languages, both the text outputs and graphical outputs can be managed.
