### Getting started with RStudio

1. What does an environment like Rstudio provide in comparison to working in the R console or running R scripts directly?
    Correct answer: It provides a well-structured history of the analyses performed
    Correct answer: It allows you to inspect data, keep a history of this inspection, and explain the transformations you perform as you go along
    Correct answer: It saves intermediate results, whether textual or graphical
    Correct answer: It allows you to generate documents in HTML, PDF or .doc(x)/.odt
    Correct answer: It allows you to ensure that a figure is the result of the computation described in the document

2. In Rstudio, what features are provided for the R language but not available for the Python language?

    Correct answer: Autocompletion on variable names
    Correct answer: The non-persistence of variables and their state
    Correct answer: For Python, only the text outputs are managed, not the graphical outputs
