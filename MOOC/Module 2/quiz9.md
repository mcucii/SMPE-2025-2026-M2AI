## Quiz 9
### The computational document: principle

- The main goal for this kind of document is to be as transparent as possible

1. What is commonly found in a computational document?

    Right answer: Commentaries
    Right answer: Code
    Right answer: An overview of data
    Right answer: Computational results 
    Right answer: Hypertext links 
    Right answer: Images
  

2. What does a computational document allow?

    Right answer: Inspect the computations
    Right answer: Easily re-run the computations if the original environment is available
    Right answer: Document the code
    Right answer: Explain why a particular computation is made based on the data analysis so far
    Right answer: Use multiple languages to perform computations (even if it may require some work)
